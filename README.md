A full-stack authentication and authorization system built by **Ashwin Wagh**.

## ✨ Features

- 🔑 **Single Sign-On** with Keycloak (OAuth 2.0 + OIDC)
- 🛡️ **Role-Based Access Control** (USER, ADMIN, MANAGER roles)
- 🔐 **JWT Token Security** for REST APIs
- 🎯 **Protected Routes** with Angular guards
- 🐳 **Dockerized** development environment
